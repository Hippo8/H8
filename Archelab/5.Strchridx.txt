 in str

Il seguente codice in C
 realizza strchridx
 (convertilo in RISC-V):
unsigned long strchridx(char *str, char c) {
    unsigned long i = 0;
    while (str[i] != '\0' and str[i] != 'c') { 
        i++;
    }
    if (str[i] == '\0') { 
        return -1;
    } else { 
        return i;
    }
}
